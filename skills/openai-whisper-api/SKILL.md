# OpenAI Whisper API
Transcribe audio files using the OpenAI Whisper API (cloud-based).

## Usage
Send audio files and this skill will transcribe them using OpenAI's Whisper model.

## Supported Formats
mp3, mp4, mpeg, mpga, m4a, wav, webm
